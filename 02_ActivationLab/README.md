## 📖 Overview

**ActivationLab** is an educational Deep Learning project that
implements and visualizes major activation functions **from scratch**
using only **NumPy** and **Matplotlib**.

It explains the mathematics behind activation functions, manual
derivatives, vanishing gradients, dead neurons, Softmax stability, and
practical use cases through rich visualizations.

This project is part of my **#100DaysOfML** journey.

## 🚀 Features

-   Sigmoid, Tanh, ReLU, Leaky ReLU, ELU, Swish, and Softmax
-   Manual derivative implementations
-   Individual activation plots
-   Comparison dashboards
-   Derivative comparison
-   Vanishing Gradient simulation
-   Dead ReLU demonstration
-   Activation heatmap
-   Softmax numerical stability
-   Cheat sheet
-   Real-world model mapping
-   Interview questions
